## 🚀 QUICK START - Hướng Dẫn Nhanh

### 1️⃣ Chỉnh Sửa Thông Tin Cá Nhân

Mở file `index.html` và tìm + sửa các phần này:

```html
<!-- Tên bạn -->
<h2 class="graduate-name">Ngô Thị Hoàng Vy</h2>

<!-- Trường học -->
<h2 class="school-name">TRƯỜNG ĐẠI HỌC NGOẠI NGỮ<br>ĐẠI HỌC ĐÀ NẴNG</h2>

<!-- Ngày tốt nghiệp -->
<p class="date">JULY <span class="number">25</span></p>

<!-- Giờ tốt nghiệp -->
<p class="time">09:30</p>

<!-- Địa điểm -->
<p class="location-name">Trường Đại học Ngoại ngữ – Đại học Đà Nẵng</p>
<p class="location-address">131 Lương Nhữ Hộc, Cẩm Lệ, Đà Nẵng</p>
```

### 2️⃣ Thêm Logo

1. Tìm file `logo.png` hoặc `logo.jpg` của trường
2. Bỏ vào folder `assets/`
3. Đó là xong!

### 3️⃣ Kiểm Tra Trên Máy

- Bấm đúp file `index.html`
- Nhấp vào hình bìa để xem mở thiệp
- Kiểm tra nó có đẹp không

### 4️⃣ Deploy Lên GitHub Pages

✅ **Cách dễ nhất - 5 phút:**

1. **Tạo GitHub Account**: https://github.com/signup
   - Email, password
   - Username: `ngo-thi-hoang-vy` (tuỳ ý)

2. **Tạo Repository**:
   - Truy cập: https://github.com/new
   - Name: `username.github.io` (thay username!)
   - Chọn "Public"
   - Create

3. **Upload Files**:
   - Click "Add file" → "Upload files"
   - Drag & drop tất cả từ folder vào
   - Commit

4. **Kích Hoạt Pages**:
   - Settings → Pages
   - Branch: main → Save
   - Đợi 2 phút

5. **Xong!**
   - Vào: `https://username.github.io`
   - Chia sẻ link này 🎉
