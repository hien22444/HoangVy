# Lễ Tốt Nghiệp - Ngô Thị Hoàng Vy

Trang web mời tham dự lễ tốt nghiệp di động và đẹp mắt.

## 📋 Cấu Trúc Project

```
project/
├── index.html          # File HTML chính
├── style.css           # File CSS styling
├── script.js           # File JavaScript
├── assets/
│   └── logo.png        # Logo trường (thay đổi tại đây)
└── README.md           # File này
```

## 🎨 Tùy Chỉnh Project

### 1. Thay đổi Tên Người Tốt Nghiệp
Mở file `index.html` và tìm dòng:
```html
<h2 class="graduate-name">Ngô Thị Hoàng Vy</h2>
```
Thay `Ngô Thị Hoàng Vy` bằng tên của bạn.

### 2. Thay đổi Thông Tin Trường
Tìm dòng:
```html
<h2 class="school-name">TRƯỜNG ĐẠI HỌC NGOẠI NGỮ<br>ĐẠI HỌC ĐÀ NẴNG</h2>
```
Thay đổi tên trường theo nhu cầu.

### 3. Thay đổi Ngày Giờ Sự Kiện
Tìm các dòng:
```html
<p class="day">THỨ BẢY</p>
<p class="date">JULY <span class="number">25</span></p>
<p class="time">09:30</p>
```
Thay đổi ngày và giờ phù hợp.

### 4. Thay đổi Địa Điểm
Tìm dòng:
```html
<p class="location-name">Trường Đại học Ngoại ngữ – Đại học Đà Nẵng</p>
<p class="location-address">131 Lương Nhữ Hộc, Cẩm Lệ, Đà Nẵng</p>
```

### 5. Thêm Logo Trường
1. Tạo folder `assets` nếu chưa có
2. Thêm file `logo.png` vào folder `assets`
3. Cập nhật đường dẫn trong HTML nếu cần

## 🚀 Deploy lên GitHub Pages (MIỄN PHÍ)

### Bước 1: Tạo GitHub Account
- Truy cập [github.com](https://github.com)
- Đăng ký tài khoản miễn phí (nếu chưa có)

### Bước 2: Tạo Repository
1. Đăng nhập GitHub
2. Click "+" ở góc trên phải → "New repository"
3. Đặt tên repository: `username.github.io` (thay `username` bằng username của bạn)
4. Chọn "Public"
5. Click "Create repository"

### Bước 3: Upload Files
1. Bấm "uploading an existing file"
2. Hoặc sử dụng Git (nếu đã cài):
```bash
git clone https://github.com/username/username.github.io.git
cd username.github.io
# Copy tất cả files từ project vào đây
git add .
git commit -m "Initial commit"
git push origin main
```

### Bước 4: Truy Cập Website
Sau 1-2 phút, website sẽ có sẵn tại:
```
https://username.github.io
```
